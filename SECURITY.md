# Security

Archangel Health processes Protected Health Information (PHI) and implements the
HIPAA Security Rule technical safeguards.

- **Security & compliance overview (for reviewers):** [`docs/security/SECURITY_OVERVIEW.md`](docs/security/SECURITY_OVERVIEW.md)
- **Full documentation set:** [`docs/security/`](docs/security/)

## Reporting a vulnerability
Please report security issues privately to **security@archangelhealth.ai** (do not
open a public issue). We aim to acknowledge within 2 business days. Responsible
disclosure is appreciated.

## For health-system security teams
For BAAs, completed HECVAT/SIG, or evidence requests, contact the Security Official
via the address above. See the overview document for the controls matrix.
